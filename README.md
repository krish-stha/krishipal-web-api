Introduction
Krishipal is a full-stack e-commerce web application developed to sell agricultural products such as seeds, pesticides, fertilizers, agricultural tools, and farming equipment. The platform provides a structured digital marketplace where farmers and agricultural stakeholders can browse products, place orders, make secure online payments, and track their purchase history.
Aim
The primary aim of Krishipal is to create a scalable and secure agricultural eCommerce platform that simplifies the process of purchasing farming-related products while ensuring efficient order and payment management.	
Objectives
Develop a RESTful API using Node.js and Express.
Implement secure payment integration using eSewa and Khalti.
Design a dynamic frontend interface using React/Next.js.
Enable downloadable invoice generation after successful payment and notify through email.
Create an admin dashboard for full system control.
Apply layered architectural principles for maintainability and scalability.
Develop a proper, fully functional eCommerce platform that connects farmers directly with customers.
